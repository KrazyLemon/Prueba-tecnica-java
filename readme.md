# API para prueba tecnica

## Pasos para ejecutar la api
    - Clonar o descargar  el proyecto
    - SI esta usando Docker solo ingrese ala carpeta donde se encuentra el 
    compose.yaml y ejecute el comando "docker compose up -d"
    - Puede probar el api desde Postman usando el link donde se encuentra
    la coleccion de request para esta api :)
 
 Postman - https://www.postman.com/j0ola1/workspace/prueba-tecnica
